# Anchor Run — Manual Test Plan

## Minimum test pass

1. Open the GitHub Pages URL.
2. Start a new run.
3. Clear Sector 1.
4. Clear at least three sectors.
5. Test pause/resume.
6. Reload page and confirm clean boot.
7. On mobile, rotate device once and return to portrait.
8. Confirm controls recover layout after rotation.

## Full acceptance pass

1. Complete all 10 sectors.
2. Record completion time.
3. Record strain events.
4. Confirm Final Access Core requires all switches.
5. Confirm no forbidden mechanics appear.
