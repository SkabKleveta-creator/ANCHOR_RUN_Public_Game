# Anchor Run — Arcade Pass B

Single-file browser arcade build for GitHub Pages.

## How to deploy on GitHub Pages

1. Create a new GitHub repository, suggested name: `ANCHOR_RUN_Public_Game`.
2. Upload these files to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set **Deploy from a branch**.
5. Select `main` and `/root`.
6. Save.

The game will run from:

```text
https://<your-github-username>.github.io/ANCHOR_RUN_Public_Game/
```

## Files

- `index.html` — the locked Anchor Run Arcade Pass B build.
- `.nojekyll` — prevents GitHub Pages/Jekyll processing.
- `README.md` — deployment notes.
