# Anchor Run — Project Lock

## Project

**Anchor Run** is the Access Layer mini-game for the Reintegration Core / Skab Digital Reality Arcade project.

## Current repository build

**Experimental 005E — Portrait Stability Cleanup**

This repository package is for **Anchor Run only**. Do not mix in Connection Trace, Drone Colony, Blackbox Squadron, FPS, or other game files.

## Scope

Anchor Run is a compact arcade route-recovery game:

- Browser first.
- Mobile portrait supported.
- Single-file playable HTML.
- GitHub Pages compatible.
- No external dependencies.

## Locked verbs

```text
MOVE
GRAPPLE
BOOST
ACT / STABILIZE / RECHARGE
```

## Never allowed

- Combat
- Inventory
- Skill trees
- Dialogue systems
- Bosses
- Jumping
- Functional ladders
- New-mechanic pile-on

## Acceptance standard

A build is not accepted because code boots. It is accepted only after device playthrough confirms the player can start, control Eli, clear sectors, and complete the route without layout or control failure.
