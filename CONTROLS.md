# Anchor Run — Controls

## Keyboard

| Input | Action |
|---|---|
| A / Left Arrow | Move left |
| D / Right Arrow | Move right |
| Space | Grapple |
| Shift | Boost |
| J / Enter | Interact / stabilize / recharge |
| Esc | Pause / resume |

## Mobile

| Touch control | Action |
|---|---|
| L | Move left |
| R | Move right |
| GRAP | Grapple |
| BOOST | Boost |
| ACT | Interact / stabilize / recharge |

## Locked control philosophy

Mobile controls are functional, not decorative. Anchor Run does not need a full D-pad because there is no up/down movement, no jumping, and no functional ladders.

The right-side button cluster remains three-button because GRAP, BOOST, and ACT are separate verbs.
